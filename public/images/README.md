# Images Directory

Place your tour images in this directory with the following names:

## Required Images:

### Hero Section Slider (used in homepage hero):
- `stock1.jpeg` - First background image (1920x1080px, landscape)
- `stock2.jpg` - Second background image (1920x1080px, landscape)
- `stock3.jpg` - Third background image (1920x1080px, landscape)

### About Section:
- `aboutme.jpeg` - Your profile photo (800x1000px, portrait orientation)

## Image Guidelines:
- Use high-quality, professional photos
- Optimize images before upload (use tools like TinyPNG)
- Recommended formats: JPEG for photos, PNG for logos
- Keep file sizes under 500KB for optimal performance
- Hero slider images should be landscape orientation for best results

## Notes:
- The hero section features an automatic image slider that smoothly transitions between images every 4 seconds
- The slider is fully automatic with no manual controls needed
- The slider overlay ensures text remains readable even with dynamic images
- The About section displays a professional portrait photo of the tour guide with parallax scroll effect
